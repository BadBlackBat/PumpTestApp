<h2>Инструкция по применению</h2>

<p>Инструкция по применению программы будет размещена здесь позже.</p>

<p>Этот текст загружается из файла <b>resources/instructions.txt</b> - чтобы
изменить содержимое инструкции, отредактируйте этот файл напрямую, менять
код программы не требуется.</p>

<p>Поддерживаются простые HTML-теги для форматирования, например:</p>
<ul>
    <li><b>жирный текст</b> - тег &lt;b&gt;...&lt;/b&gt;</li>
    <li><i>курсив</i> - тег &lt;i&gt;...&lt;/i&gt;</li>
    <li>заголовки - теги &lt;h2&gt;...&lt;/h2&gt;, &lt;h3&gt;...&lt;/h3&gt;</li>
    <li>списки - теги &lt;ul&gt;/&lt;li&gt;</li>
</ul>
